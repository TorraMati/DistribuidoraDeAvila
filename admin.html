☁️ Configurar base de datos (Supabase)

Gratis para esta tienda. Una vez configurado, todos los dispositivos ven el mismo catálogo.

1
Creá un proyecto en Supabase

Entrá a supabase.com → New project. Anotá la contraseña de la base de datos.

2
Creá la tabla de productos

En el panel → SQL Editor → New query → pegá esto y ejecutá:

create table if not exists products (
  id text primary key,
  codigo text,
  nombre text not null,
  categoria text,
  precio float8 default 0,
  stock text,
  imagen text
);

alter table products enable row level security;
create policy "read_all"   on products for select using (true);
create policy "insert_all" on products for insert with check (true);
create policy "update_all" on products for update using (true);
create policy "delete_all" on products for delete using (true);
3
Creá el bucket de imágenes (para fotos)

Storage → New bucket → nombre: product-images → marcá Public bucket → Create.

4
Copiá las claves del proyecto

Settings → API → copiá Project URL y anon public key.

5
Pegá las claves en js/supabase-config.js

Abrí ese archivo y reemplazá PEGAR-AQUI con los valores copiados. Guardá y subí a GitHub.

⚠️ Zona peligrosa

Eliminá todos los productos del catálogo. No se puede deshacer.
