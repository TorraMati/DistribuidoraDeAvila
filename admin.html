☁️ Configurar base de datos (Firebase)

Para que el catálogo sea visible desde cualquier dispositivo, necesitás conectar Firebase. Es gratis para el uso de esta tienda.

1
Creá un proyecto en Firebase

Entrá a console.firebase.google.com con tu cuenta Google y hacé clic en "Agregar proyecto".

2
Activá Firestore Database

En el menú lateral → "Firestore Database" → "Crear base de datos" → modo Producción → elegí ubicación southamerica-east1.

3
Configurá las reglas de seguridad

En Firestore → pestaña "Reglas" → reemplazá todo con:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /catalog/{doc} {
      allow read: true;
      allow write: true;
    }
  }
}

Publicá las reglas.

4
Obtené la configuración

En el proyecto → ⚙️ Configuración → "Tus apps" → hacé clic en "</>" (web) → registrá la app → copiá el objeto firebaseConfig.

5
Pegalo en el archivo js/firebase-config.js

Abrí ese archivo y reemplazá los valores PEGAR-AQUI con los de tu proyecto.

⚠️ Zona peligrosa

Eliminá todos los productos del catálogo. Esta acción no se puede deshacer.
